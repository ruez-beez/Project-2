# Crochet Tutorial 

## Project Description
This is a simple web interface meant to display the varying steps of this amigurumi pattern. 

## Intended User or Player
This interface is designed for crochet beginners or anyone interested in a beginner crochet pattern. 

## User / Player Goal
Users should be able to scroll through a series of steps or pick which section they want to go to from the side navigation.

## Inspiration Interfaces
Solaria Webspace’s Homepage
I was inspired by their side navigation as well as how they made their layout responsive. Their use of visual feedback that was in theme with the design of the website was an aspect I wanted to incorporate. 

PetraPixel’s Javascript Tutorial
This was another main inspiration. From looking at this website's code I learned how useful grids can be in creating any type of layout.  Although I did not use grid areas like them, I still incorporated grids into my project. 

## Design Choices
- Layout:  Web layout with a side bar navigation and main content divided by sections.
- Color: Red, white, greens. 
- Typography: Glass Antiqua, Nunito Sans
- Visual hierarchy:  Title first, then main content, then navigation.
- Feedback states:  Navigation links darken in color when hovered over. Regular links will turn red and increase in font weight when hovered over. In their normal state regular links will appear black and with an underline.
- Responsive design: When the screen reaches a certain size the side navigation will disappear and appear only when a button on the top left corner is clicked or hovered over. Text and the image will also grow or shrink in size depending on screen width.
